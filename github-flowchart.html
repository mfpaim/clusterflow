AI Lesson Plan Generator Workflows

Comparison of automation complexity: Simple, Standard, and Advanced versions.

Simple Workflow

1️⃣ Google Drive Trigger

Detects new PDF uploads from teachers.

↓

2️⃣ Extract PDF Text

Converts uploaded PDF to plain text.

↓

3️⃣ Generate Lesson Plan (AI)

Uses AI to create a lesson plan directly from text.

↓

4️⃣ Send Notification

Emails teacher with AI-generated plan summary.

↓

5️⃣ Log Success

Records workflow completion status.

Standard Workflow

1️⃣ Google Drive Trigger

Watches for new lesson-related PDF uploads.

↓

2️⃣ Extract PDF Text

Extracts all text for AI analysis.

↓

3️⃣ Extract Metadata

Reads file name, date, grade, and subject.

↓

4️⃣ Generate Lesson Plan (AI)

Creates structured lesson plan content using metadata.

↓

5️⃣ Create Google Doc

Formats plan and saves it to Google Drive.

↓

6️⃣ Send Notification

Sends the document link to the teacher.

↓

7️⃣ Log Success

Registers completion in workflow logs.

Advanced Workflow

1️⃣ Google Drive Trigger

Monitors multiple shared folders for new test uploads.

↓

2️⃣ Extract PDF Text

Extracts question and answer structures from tests.

↓

3️⃣ Extract Metadata

Detects filename, grade, subject, and test date.

↓

4️⃣ Standards Lookup

Fetches matching educational standards (Common Core, NGSS, etc.).

↓

5️⃣ Merge Data

Combines standards, metadata, and extracted text for the AI prompt.

↓

6️⃣ Generate Lesson Plan (AI)

Creates a detailed, standards-aligned lesson plan using AI.

↓

7️⃣ Create Google Doc

Generates a formatted document ready for classroom use.

↓

8️⃣ Send Notification

Notifies teacher with the Google Docs link.

↓

9️⃣ Log Success

Logs the result and runtime data to analytics sheet.
